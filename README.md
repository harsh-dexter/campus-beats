# Campus Beats

A modern, scalable Next.js 14+ application designed with a mobile-first, dark-themed UI.

## Technologies Used
- **Next.js 14+ (App Router)**
- **TypeScript**
- **Tailwind CSS**

## Folder Structure
- `/app`: Next.js App Router root
  - `/(auth)`: Auth-related pages (login, register)
  - `/(main)`: Main application views
  - `/api`: API routes
- `/components`: Reusable UI components
- `/lib`: Utility functions and third-party setups
- `/hooks`: Custom React hooks
- `/store`: State management (Zustand, Redux, Context, etc.)

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
